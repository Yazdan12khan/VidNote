How to use — Quick:
  1. Paste a YouTube link or direct .mp4 URL → Load (Enter also works).
  2. Use buttons or keyboard shortcuts (Shift+Space play/pause, Shift+→ forward, Shift+← back).
  3. Toggle Start Logging to auto-insert timestamped actions into Notes.
  4. Click Insert time in the Notes column to insert current time at cursor.
  5. Export notes as .txt / .md / .json from the Notes panel.
